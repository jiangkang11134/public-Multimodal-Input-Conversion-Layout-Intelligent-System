# 版图制作者智能体 (Pattern Maker Agent)

## 一、概述

版图制作者智能体负责将设计师智能体提取的设计特征转换为可制造的服装版图（纸样）。它是连接设计意图与实际生产的关键环节。

## 二、核心功能

### 2.1 DSL推理引擎

将设计特征映射为版图参数：

- 输入：设计特征列表（如 ["meta__upper__Shirt", "shirt__length__long"]）
- 处理：使用 lmm_utils.projector 进行参数投影
- 输出：结构化的版图参数

### 2.2 版图生成器

使用 pygarment 库生成服装版图：

- 输入：版图参数（YAML格式）
- 处理：调用 garment_programs 生成服装组件
- 输出：pattern_specification.json + 2D版图图像

## 三、技术实现

### 3.1 参数投影

基于 lmm_utils.projector 的 input_caption2random_default_cption 方法：

- 将设计特征填充为完整的设计描述
- 生成符合模板格式的设计参数
- 处理缺失参数的默认值填充

### 3.2 版图生成

使用 garment_programs.meta_garment 模块：

- 支持多种服装类型：T恤、衬衫、裙子、裤子、外套等
- 自动处理组件拼接和连接关系
- 生成标准的 pattern JSON 格式

## 四、工作流程

`
设计特征 → DSL推理 → 版图参数 → 版图生成 → pattern JSON + 图像
`

## 五、输出格式

### 5.1 pattern_specification.json

包含完整的版图规格信息：
- 面板（panels）定义
- 边缘（edges）定义
- 连接器（connectors）定义
- 组件（components）定义

### 5.2 2D版图图像

使用 Cairo 渲染的服装纸样图像：
- 包含所有面板和标记
- 支持多种输出格式（PNG、SVG）

## 六、技术栈

| 模块 | 技术 |
|------|------|
| 参数投影 | lmm_utils.projector |
| 版图引擎 | pygarment（Design2GarmentCode） |
| 图像渲染 | Cairo |
| 服装程序 | garment_programs |

## 七、核心文件

| 文件 | 功能 |
|------|------|
| pattern_maker_agent.py | 版图制作者智能体主类 |
| dsl_infer.py | DSL推理引擎 |
| executor.py | 版图执行器 |
| integrated_pipeline.py | 整合管道（执行引擎） |
| run_full_pipeline.py | 完整管道执行 |

## 八、使用示例

`python
from intelligent_agents.pattern_maker import PatternMakerAgent

agent = PatternMakerAgent()

json_path = agent.process_design(
    design_features=["meta__upper__Shirt", "shirt__length__long"],
    image_path=None,
    output_dir="./output/demo",
    debug_trace=True,
)

print(f"版图生成成功: {json_path}")
`

## 九、依赖关系

版图制作者智能体依赖执行引擎提供的核心能力：
- pygarment 服装代码库
- garment_programs 服装程序库
- assets 资源文件（人体模型、设计参数模板）

## 十、总结

版图制作者智能体是系统的核心生产环节，它将抽象的设计特征转化为具体的、可用于实际生产的服装版图。通过 DSL推理 + 版图生成的组合方式，实现了从设计意图到生产数据的自动化转换。
