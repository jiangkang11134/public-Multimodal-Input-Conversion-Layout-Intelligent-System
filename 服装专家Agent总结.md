# 服装专家 Agent - 完整项目总结

## 一、项目概述

服装专家 Agent 是一个基于检索增强生成（RAG）和网络爬虫技术的智能服装知识问答系统。该系统融合了两大核心能力：

1. **知识问答**：基于服装面料知识库，回答用户关于面料、工艺、穿搭等专业问题
2. **潮流预测**：通过爬虫采集各大购物与社交平台的爆款服装数据，进行潮流分析和预测

---

## 二、系统架构

服装专家 Agent 包含三个层次：前端交互层、核心业务层、数据采集层。核心业务层包含文档解析、智能分块、混合嵌入、混合检索引擎和问答生成器。数据采集层包含小红书、淘宝、微博、知乎爬虫和潮流分析模块。

**⚠️ 当前状态**：爬虫软件尚未接入服装专家 Agent，两个系统独立运行。

---

## 三、核心功能模块

### 3.1 知识问答系统（RAG）

| 模块 | 文件 | 功能描述 |
|------|------|----------|
| 文档解析 | work/document_doing.py | 支持 PDF/DOCX/PPTX 解析 |
| OCR识别 | work/ocr.py | PaddleOCR + Qwen-VL-OCR 双重识别 |
| 智能分块 | work/spliter.py | 语义分块，支持表格、图片、问答等类型 |
| 混合嵌入 | work/embedding.py | BGE-M3 稠密+稀疏向量生成 |
| 混合检索 | work/retriever.py | 稠密+稀疏检索 + BGE-reranker 重排序 |
| 问答生成 | work/qa.py | 基于检索结果生成精准回答 |
| 配置管理 | work/config.py | 模型路径、API密钥、分块参数等配置 |

### 3.2 数据采集系统（爬虫）

| 模块 | 文件 | 功能描述 |
|------|------|----------|

| 淘宝爬虫 | 女装牛仔裤爬虫脚本/2_淘宝.py | 爬取热销商品、图片、销量、价格数据 |

## 四、技术栈

### 4.1 RAG系统技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 向量模型 | BGE-M3 | 稠密+稀疏混合嵌入 |
| 重排序模型 | BGE-reranker-base | 结果重排序 |
| 向量数据库 | ChromaDB | 轻量级本地向量数据库 |
| OCR引擎 | PaddleOCR | 百度飞桨OCR |
| 大模型 | Qwen-VL-OCR / Qwen-VL-Max | 阿里云通义千问 |

### 4.2 爬虫系统技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 浏览器自动化 | Playwright | Chromium引擎 |
| GUI框架 | Tkinter | 内置GUI库 |
| 图片处理 | PIL/Pillow | 图片下载和展示 |

## 五、数据处理流程

### 5.1 文档入库流程

文档输入 → 文本提取 → OCR识别 → LLM兜底 → 排序合并 → 分块处理 → 文本清洗 → 混合嵌入 → 向量入库

### 5.2 检索问答流程

用户提问 → 混合检索（稠密+稀疏）→ 结果融合 → BGE-reranker重排序 → 答案生成

### 5.3 爬虫采集流程

关键词配置 → 浏览器初始化 → 登录验证 → 搜索采集 → 数据解析 → 图片下载 → CSV导出 → 潮流分析

## 六、核心算法与策略

### 6.3 爬虫采集策略

## 十二、总结

服装专家 Agent 是一个集知识问答和潮流预测于一体的智能系统，具有以下特点：

1. **完整的RAG架构**：从文档解析到智能分块、混合检索、答案生成，形成完整的知识问答闭环
2. **优秀的评估指标**：上下文精确率0.9474，召回率0.8772，F1分数0.9109，达到行业优秀水平
3. **多平台数据采集**：支持小红书、淘宝等平台的爆款数据采集，为潮流预测提供数据支撑
4. **混合检索策略**：结合稠密向量语义匹配和稀疏向量关键词匹配，提升检索准确性
5. **智能分块技术**：基于语义的文本分块，保留表格、问答等完整语义单位
6. **可视化管理界面**：Tkinter爬虫管理界面，方便用户操作和查看数据

**当前状态**：RAG知识问答系统已完善并达到优秀评估指标；爬虫系统独立运行，尚未接入服装专家Agent。下一步需要完成两个系统的集成，实现完整的潮流预测能力。

该系统可应用于服装行业的智能客服、潮流分析、商品推荐等场景，具有广阔的应用前景。
