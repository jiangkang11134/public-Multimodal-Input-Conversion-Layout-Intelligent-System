# 设计师智能体 (Designer Agent)

## 一、概述

设计师智能体负责解析设计输入，提取设计特征。它是系统的前端智能体，支持两种输入方式：文本描述和草图图像。

## 二、核心模块

### 2.1 文本设计模块

基于大模型 API + 提示词强约束实现：

#### 输入输出
- **输入**：自然语言描述（如"一条高腰A字中长裙，配合修身上衣"）
- **输出**：结构化设计特征列表

#### 技术方案
- **模型**：阿里云通义千问 API（qwen2.5-72b-instruct）
- **方法**：精心设计的提示词模板 + 强约束输出格式
- **支持任务**：文本设计、设计修改、压力图放松

#### 处理流程
`
自然语言输入 → 提示词构建 → API调用 → 输出解析 → 设计特征列表
`

#### 输出示例
`
- meta__upper__FittedShirt
- shirt__length__long
- collar__f_collar__CircleNeckHalf
- sleeve__length__full
`

#### 核心文件
| 文件 | 功能 |
|------|------|
| designer_agent.py | 设计师智能体主类 |
| mmua_designer_agent.py | MMUA适配器，封装多模态能力 |
| design_feature_converter.py | 设计特征转换器 |

### 2.2 草图识别模块

基于微调视觉大模型实现：

#### 输入输出
- **输入**：正面渲染图 + 背面渲染图
- **输出**：结构化设计属性列表

#### 处理流程
1. **读取服装渲染图** → 正面 + 背面视图
2. **去除人体区域** → HSV颜色特征分割，保留服装主体
3. **草图风格转换** → 灰度化、边缘检测、手绘风格增强
4. **前后视图拼接** → 合成统一训练/推理样本
5. **属性识别** → 微调 Qwen2.5-VL 模型输出结构化属性

#### 技术栈
| 模块 | 技术 |
|------|------|
| 基础模型 | Qwen2.5-VL-7B |
| 微调方法 | LoRA 参数高效微调 |
| 图像预处理 | OpenCV + 形态学处理 |

#### 关键步骤详解

##### 2.2.1 人体区域去除
使用 HSV 颜色特征识别人体区域：
- 明亮度低的像素
- 饱和度低的像素
- 灰度类像素
- 白色背景像素

通过这些条件组合出"身体区域掩码"，将人体部分从图像中移除。

##### 2.2.2 草图风格转换
把服装图像转成"手绘素描/草图感"的视觉效果：
1. 转成灰度图
2. 双边滤波平滑噪声，保留结构边缘
3. Canny边缘检测提取线条
4. 叠加线条到原图，形成手绘轮廓感
5. 降低颜色饱和度，偏"铅笔草图"
6. 统一背景为白色，增强清晰度

##### 2.2.3 前后视图拼接
将正反面合成一张整体图：
1. 裁剪服装主体，去掉多余空白
2. 等比例缩放和居中放置
3. 前视图放左半区，后视图放右半区
4. 中间留白色分隔线
5. 输出统一尺寸的合成图

##### 2.2.4 模型训练
使用 LoRA 微调 Qwen2.5-VL：
- 训练样本：前视图 + 后视图 + 属性标签
- 训练目标：输出结构化属性列表
- 优势：训练成本低、显存需求小、小数据集上效果好

#### 输入输出示例
`
输入：正面渲染图 + 背面渲染图
输出：
- meta__upper__FittedShirt
- shirt__length__long
- collar__f_collar__CircleNeckHalf
- sleeve__length__full
`

## 三、使用示例

### 3.1 文本设计

`python
from intelligent_agents.designer import DesignerAgent

agent = DesignerAgent()

result = agent.process_design({
    "type": "text",
    "content": "一条高腰A字中长裙，配合修身上衣"
})

print(result["design_features"])
`

### 3.2 草图识别

`python
from intelligent_agents.designer import DesignerAgent

agent = DesignerAgent()

result = agent.process_design({
    "type": "image",
    "front_image": "front_view.jpg",
    "back_image": "back_view.jpg"
})

print(result["design_features"])
`

## 四、优缺点分析

### 优点
- 支持多模态输入（文本 + 图像）
- 文本模块响应快速，无需训练
- 草图模块精度高，基于微调模型
- 输出结构化，便于后续处理

### 限制
- 草图模块依赖高质量训练数据
- 图像预处理依赖手工设定的颜色阈值
- 复杂细节识别仍有难度

## 五、总结

设计师智能体是系统的"眼睛"和"耳朵"，它能够：
- 听懂用户的自然语言描述（文本设计模块）
- 看懂服装的草图图像（草图识别模块）
- 将两种输入统一转换为结构化的设计特征

这为后续的版图生成提供了准确、统一的输入数据。
